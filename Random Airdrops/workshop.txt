title=Random Aidrops
description=This mods brings the random airdrops to your server, airdrops can be configured setting the:
description=- Hours to remove
description=- Chance to spawn every hour
description=- Performance Tickrate
description=- Loot table (Lua Folder)
description=- Spawn Coordinates (Lua Folder)
description=Features:
description=- Airdrops never spawn above each other
description=- Airdrops doenst despawn if player is close
description=- Message to player chat when the airdrop is spawned + area name (not work in singleplayer because in singleplayer doenst have chat)
description=- Air plane sound when the air drop spawn
description=- If the servers close, old airdrops will be deleted automatically (configurable)
description=- Fully customizable loot table (see the github wiki)
description=- Fully customizable spawn coordinates (see the github wiki)
description=
description=How it works?
description=
description=- The server random a chance if the chances hits ok then the airdrop coordinates is stored in "cache"
description=- if the cache have airdrops then the server starting ticking every 30 ticks to check if there is a player to spawn the airdrop (vehicles cannot spawn without a chunk been loading) in that specific position, also when the airdrop is created is removed from the cache
description=- if the chunk in that specific position is loaded then the airdrop will be created
description=- when the timer goes below the 0 the airdrop will be removed from the world and from the cache if not created yet, unless the player is loading the chunk of the airdrop
description=- when the server starts all old airdrops will be despawned when the chunk loads
